# VCS Project

A simple project created for the Version Control Systems (VCS) comprehensive assignment.  
It demonstrates Git basics, branching, merging, conflict handling, documentation, issues, milestones, and GitHub Actions.

## Project Overview
This project is developed step-by-step across four phases, showing how Git and GitHub workflows evolve in real development.

## Installation
1. Clone this repository:
2. Open the project folder.
3. Run the example file (if applicable).

## Features
- Clean folder structure (src/, docs/, assets/)
- Branching and merging
- Merge conflict creation & resolution
- GitHub Pull Request demonstration
- Issues, labels & milestones
- GitHub Pages / Wiki
- GitHub Actions CI workflow
- AI/Copilot assisted code

## Usage
Navigate through the code inside the `src/` folder.  
Documentation is available inside the `docs/` folder or on GitHub Pages.
